
# The Pattern Habit

A single-page habit tracker built for polymaths- track daily habits across custom domains, build streaks, and map cross-domain connections.

## How to run

No build step. No dependencies.

```bash
git clone <your-repo-url>
cd habit-tracker
open index.html   # macOS
# or just double-click index.html in your file explorer
# or simply use the deployed URL

```

Everything runs in the browser while data is handled via localstorage.
